ZeptoN

For a current project, a new programming language "ZeptoN" from "zepto" + "N". A small programming language that is the Nth programming language (as programming languages go from 1 to N) where this is the Nth language.

ZeptoN extends Java by adding a program entity construct like a class (without the object-oriented trimmings), with an implicit program entry point as a block. Yet ZeptoN is completely Java syntax and concept compatible. It puts the program back into Java programming!

ZeptoN is now available on my GitHub repo for download, or the GitHub page. This is the ZeptoN "Echo" trans-compiler which transpiles to Java, and then uses the Java Compiler API to build a Java bytecode .class file.

ZeptoN Code Example...


   prog helloWorld {
   begin
      println("Hello, World!!!");
      exit(0);
   }

Research Papers
