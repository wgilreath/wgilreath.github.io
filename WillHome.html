Specific Projects:

  • KVaaS - Java library for using the key-value as a service (api.keyvalue.xyz) online web data store.
  • Boz - compiler for a programming language developed using JavaCC.

  • XML Tokenizer - XML tokenization tags, attributes in Java.
  • Binar Shuffle/Binar Sort - shuffle (unsorting) and sorting by using bits of binary datum in Java.
  • FunCL - interpreter for the FUNctor Clause Language, a functional style programming language that is both like Forth (using explicit stack management) and LISP (functions operate on functions) but without the reverse Polish notation (RPN) or parenthesis in Java.

    The "Hello World!!!" program in FunCL is:
DEF HELLOWORLD
    PUSH EOLN PUSH "Hello World!!!" ECHO ECHO .
    HELLOWORLD .
.


Technical Writings
            
  • Computer Architecture: A Minimalist Perspective - a computer architecture textbook that explores ultimate minimalism of a one-instruction set computer--a computer architecture with only one-instruction.
  • XString - a method of "flattening" an XML document into a linear string; then an XML document can be processed as a string. Interestingly, one could embed XML within an XML document using an XString.
  • Hash Sort: A Linear Time Complexity Multi-Dimensional Sorting Algorithm - a sorting algorithm that uses hashing within a N x N matrix.
  • Binar Sort: A Linear Generalized Sorting Algorithm - a sorting algorithm that partitions like a QuickSort but uses extracted bits from the datum element to sort the data into an ordered permutation.
  • Binar Shuffle: Shuffling Bit by Bit - a shuffle algorithm that uses a "bit schedule" to shuffle each datum element into a random permutation.
  • Mynx Programming Language - the user manual for a programming language I developed that is Pascal/Algol-style syntax, but an object-oriented programming language.
  • Boz Programming Language - the user manual for the the Boz programming language that I developed that is a "duck typed" object-oriented programming language.
